# 简介 
我们用pandas读xls文档，把读好的整个资料筛选出，然后把想要的资料写入和读入一个新的tsv文档，把tsv文档供webapp使用，其中我们也用到了把tsv转化为csv。
		

## 输入：
用户输入变数1:课程内容（型态：下拉）
## 输出：
用户得到输出结果为：上限人数
## 从输入到输出，本组作品使用了：
